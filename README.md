# Crown Melbourne — Kitchen Equipment Intelligence

A unified analytical dashboard for work order performance, maintenance economics, and equipment health across Crown Melbourne F&B kitchens.

## 🌐 Live Site

**[https://praneshkumar-r.github.io/Crown/](https://praneshkumar-r.github.io/Crown/)**

## How to Use

1. Visit the live site
2. Drop your Excel data files (`6 Months Data.xlsx` and `6 month WO asset cost .xlsx`)
3. The dashboard renders automatically with interactive charts and analytics

> All processing happens in-browser. No data leaves your device.

## Features

- **Executive View** — KPIs, monthly trends, priority breakdown
- **Maintenance Analytics** — CM vs PM analysis, resolution velocity
- **Location Hotspots** — Cost by venue, top locations
- **Cost Intelligence** — Spend distribution, asset cost exposure
- **Equipment Breakdown** — Equipment type analysis from WO descriptions
